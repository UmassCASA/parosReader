# parosReader
Repository for the CASA Infrasound Lab

# Dependencies
* git

```
apt-get update
apt-get install git gcc screen
```

# Installing the Code
You can compile the code with:
```
git clone https://github.com/UmassCASA/parosReader.git
cd parosreader/src
chmod +x run.sh
```

# Running the Program
Execute the following commands as root
```
./run.sh
```
